# Assets フォルダ

このフォルダには、LPで使用する画像アセットを配置します。

## 必要なファイル

| ファイル名 | 用途 | 推奨サイズ |
|----------|------|----------|
| `logo.png` | ヘッダー・フッターのロゴ | 高さ80px以上の正方形 or 横長PNG（透過推奨） |
| `og-image.png`（任意） | SNSシェア時のOGP画像 | 1200×630px |

## ロゴ配置手順

1. 「イベントカルテ」ロゴPNG画像を本フォルダ（`assets/`）に `logo.png` として保存
2. `git add assets/logo.png && git commit -m "ロゴ画像追加" && git push`
3. GitHub Pages反映後、自動的にヘッダーに表示されます

## 画像がない場合

`index.html` は `logo.png` が無い場合、SVGで描画されたフォールバックロゴ（クリップボード+チェックリスト風）を表示します。
画像配置までの暫定表示として機能します。
